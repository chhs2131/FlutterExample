## Firebase SDK 설치
### Firebase-tools 설치
npm이 있다면 이를 이용해서 설치
`npm install -g firebase-tools`

설치완료 후 firebase 로그인
`firebase login`
![img.png](img.png)

FlutterFire CLI 설치
- 윈도우는 환경변수를 직접등록하라는 멘트와 함께 경로 정보가 나오니 집중해서 봐야한다!!
`dart pub global activate flutterfire_cli`
